Tag Practice

The HTML Tags. Not the game of tag.
  • p tag (p): used to create paragraphs in HTML code.
  • ordered list (ol): used to create a list where the order matters. Can be numbered with Latin or Roman numerals.
  • emphasis (em): often known as ITALIC, used to shear text.
  • heading tags (h#): used to create larger heading texts.
  • table tag (table): used to create a table on the page.
